# Changelog

All notable changes to `research-tools` are documented here.
Format: [Keep a Changelog](https://keepachangelog.com/en/1.1.0/); versioning: SemVer.

## [1.0.0] - 2026-07-12

### Added

- Initial release: the `researcher` agent extracted from the DevDigest
  harness — read-only project/internet research with strict report templates,
  interview mode for ambiguous requests, and honesty rules.

### Changed

- Report templates translated to English; the agent now responds in the
  user's language instead of a hardcoded one.
- Added an explicit Inputs section (question, optional mode and scope hints).
